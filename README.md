# kaggle-env

This repo dedicates for conda python env set up on Kaggle

## Usage

Copy this notebook from Kaggle: [URL](https://www.kaggle.com/code/hoanggvo/change-python-version-kaggle-v2-taylor-amarel)

```bash
git clone https://github.com/HoangVo-Prog/kaggle-env.git
bash kaggle-env/setup.sh 3.10 # Python 3.10
python --version # 3.10
which python # /opt/conda/envs/newCondaEnvironment/bin/python
```
